Half-Century Later:
From Sketchpad61 to Sketchpad14

Hesam Samimi

In 1961, Ivan Sutherland built a computer program called Sketchpad. It was the first in many things: first graphical user interface (GUI), first time an oscilloscope was modified to be used as a computer monitor as we know it now, first gesture UI using a light pen, first computer drawing system. It was also first in categories not commonly known. To most, it appears to be a CAD drawing tool, something like the great-great grandpa of AutoCAD. Programming languages pioneer Alan Kay points out that that it is really the first object oriented system as well as first fully declarative programming system.
